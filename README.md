# VisionCraft – AI-Powered Retail Creative Builder

VisionCraft is an AI-driven web application that generates retailer-compliant and professional ad creatives. Users can upload product images, describe their design style, and instantly receive polished creatives suitable for online marketplaces and social media platforms.

## Tech Stack
- React + TypeScript – Component-based UI framework
- Vite – Fast build tool
- Tailwind CSS – Utility-first styling
- Google Gemini AI – Creative generation and background design
- Netlify / Vercel – Deployment and hosting

## Key Features
- Upload product photos and generate AI-powered creatives
- Multiple ad formats (1:1, 9:16, 4:5, etc.)
- Clean and responsive user interface
- Compliance-friendly layouts for retailers
- Downloadable creative outputs

## Installation

Clone the repository:
git clone https://github.com/krishnaveniammi/VisionCraft-AI-Retail-creative-Builder.git

cd VisionCraft-AI-Retail-creative-Builder
npm install
npm run dev


### Add Gemini API Key
Create a `.env.local` file in the project root and add:
VITE_API_KEY=your_api_key


## Deployment

Build the project for production:
npm run build


Deploy the `dist` folder using  Vercel.
